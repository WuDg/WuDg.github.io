Java测试

压力测试👏

ab(apache benchmark)安装🐱‍🏍

ab测试的命令参数🤷‍♀️

ab [可选的参数选项] 需要进行压力测试的url🐱‍🐉

模拟并发请求100次，总共请求10000次

测试结果： 请求吞吐量为：180.04/s, 单个请求处理时间为：55.543ms

分析结果： 请求吞吐量为：511.34/s, 服务器处理单个请求的时间为19.556 ms。

注意事项👀

  • 测试机与被测试机要分开
  • 不要对线上的服务器做压力测试
  • 观察测试工具ab所在机器，以及被测试的前端机的CPU、内存、网络等都不超过最高限度的75%
